# Output contracts (shared)

Use these contracts to standardize skill outputs.

## Evidence-first contract
- Result
- Evidence (anchors)
- Confidence
- Assumptions
- Next actions

## Comparison contract
- Options
- Criteria
- Comparison
- Recommendation
- Tradeoffs
- Confidence

## Verification contract
- Verdict (PASS/FAIL)
- Checks run
- Evidence
- Failures
- Fix suggestions

## World-model contract
- Structured model (JSON/YAML)
- Variables/entities
- Rules/assumptions
- Evidence anchors
- Uncertainty
