# World modeling atoms (real + digital)

If you want Claude Code skills to model the real and digital world, the core missing capability set is:

## 1) State
- world-state
- temporal-reasoning
- spatial-reasoning

## 2) Dynamics
- state-transition
- causal-model
- simulation

## 3) Identity & continuity
- identity-resolution
- provenance
- grounding

## 4) Uncertainty
- uncertainty-model
- estimate-* (DIS)
- forecast-* (DIS)

## 5) Action + control
- plan
- schedule
- act-plan
- rollback

These atoms let you represent:
- what exists now
- how it changes
- why it changes
- how sure you are
- what you did about it
